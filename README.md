StudyCraft AI is an AI-powered study roadmap planner that helps students create personalized, day-wise study schedules based on their exam, subjects, available study hours, duration, and difficulty level.

## Features
- AI-generated study roadmaps
- Day-wise structured plans
- Clean and simple UI
- Google login using Firebase
- Secure backend with Node.js and Express

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- Firebase Authentication
- Gemini API
